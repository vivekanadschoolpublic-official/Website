# Website

This website is made to attract more customers and get more clients.
Uses the following technology:

- Flask; as the need was for fast and reliable deployment.
- PostgreSQL with google account and analytics integration; traffic monitoring
- A NoSQL framework to maintain the loose records.
- will also need to use the following packages.
  - `Flask-SQLAlchemy`,`Flask-Migrate`,`Flask-Cors` and `Flask-WTF`.
